Промпт для генерации веб-приложения "Примерка причесок"

Нужно создать веб-приложение для онлайн-примерки причесок.

Технологии

HTML + TailwindCSS

Каждая страница реализуется как отдельный HTML-файл, но подгружается в главную страницу (по факту переключается только видимость).

Логика переключения страниц — на клиенте (без перезагрузки).

API

Уже готова модель (нейронка), доступная через API:

POST-запрос получает 2 URL (фото пользователя и фото прически).

Возвращает URL результирующего фото.

Страницы

Главная (неавторизованный пользователь):

Краткое описание сервиса.

Пример работы (одновременно показаны 3 фото: исходное фото пользователя, фото прически, результат).

Кнопки регистрации/авторизации:

через Google, VK, Yandex, Mail

через внутреннюю регистрацию

Страница примерки (после входа):

Возможность загрузить и просмотреть превью двух фото:

Фото человека, на кого примеряют прическу

Фото прически

Кнопка «Примерить прическу» → отправляется запрос к API.

Индикатор загрузки при ожидании ответа от API.

Отображение результата.

Возможность загружать свои модели причесок.

История:

Список всех прошлых примерок пользователя.

Для каждой примерки: дата, исходные фото, результат.

Возможность удалить запись из истории.

Профиль пользователя:

Имя или email, тип авторизации.

Количество всех примерок + дата последней.

Возможность загружать свои модели причесок.

Возможность группировать модели по категориям (короткие, длинные и т. д.).

UI/UX

Адаптивный дизайн (десктоп + мобильные устройства).

Минималистичный стиль, ориентированный на эстетику iPhone.

Не указывать конкретные цвета, размеры или отступы — дать свободу в выборе стилей.

Обеспечить чистый и современный интерфейс.
